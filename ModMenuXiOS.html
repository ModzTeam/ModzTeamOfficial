Installation

Before you start

You need a JAILBROKEN device to install this mod, if you dont know what a jailbreak is go look it up. I should'nt need to tell you.

Install steps

For everyone else with a jailbroken device, all you need to do is tap the download button and add the repo to whatever package manager you have. (sileo, cydia, installer and zebra).
Then just tap on my source and install the mod.
You will need a key to access the mod. To get this key all you gotta do is join the discord server and open a ticket with your id in the Get Key channel. Then just paste it into the menu popup and enjoy.
